## 四、字段命名规范

| 字段类型 | 前缀/后缀 | 示例 |
|---------|----------|------|
| 主键 | Id | UserId, FlowId |
| 外键 | xxxId | TenantId, FlowInfoId |
| 名称 | Name | FlowName, SuiteName |
| 编码 | Code | FlowKey, SuiteCode |
| 描述 | Desc | FlowDesc, ObjectDesc |
| 状态 | Status | Status, LoginStatus |
| 时间 | At/Time | CreatedAt, UpdatedAt, LastRunTime |
| 数量 | Count | TriggerCount, RunCount |
| JSON数据 | Json | FlowContent, InputJson |
| 布尔值 | Is/Flag | IsDeleted (用 Deleted 代替) |

## 五、软删除与租户隔离

### 5.1 软删除
- 所有业务表均包含 `deleted` 字段
- `deleted = 0` 表示正常数据
- `deleted = 1` 表示已删除（逻辑删除）

### 5.2 租户隔离
- 所有业务表均包含 `tenant_id` 字段（继承自 BaseEntity）
- `tenant_id = null` 表示全局数据（不隔离）
- `tenant_id` 有值时，查询自动按租户过滤
